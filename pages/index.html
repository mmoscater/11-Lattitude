Summary: Lattitude vs. X

The purpose of this project was to analyze how weather changes as you get close to the equator. To acomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. lattitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visulaizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
